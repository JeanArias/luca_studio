# Luca Studio — Sitio Web Oficial

[![Luca Studio](https://img.shields.io/badge/Luca%20Studio-Sitios%20Web%20que%20venden- C8F135)](https://luca-studio.com)

Repositorio oficial del sitio web de **Luca Studio**, un estudio de diseño y desarrollo web especializado en crear presencia digital de alto impacto para negocios, artistas y emprendedores en Costa Rica.

## ✨ Características Principales

- **Diseño Brutalista Editorial**: Estética única y funcional, alejada de plantillas genéricas.
- **Rendimiento Extremo**: Código limpio, imágenes optimizadas y hosting en Cloudflare.
- **SEO Integrado**: Arquitectura de venta, schema markup y etiquetas meta listas para rankear.
- **Animación de Intro**: Logo con efecto "Bloom" y escritura de "Luca Studio".
- **Cursor Personalizado**: Cursor con efecto de expansión para una experiencia inmersiva.
- **Totalmente Responsive**: Adaptado a móviles, tablets y escritorio.

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, TailwindCSS, CSS3 (Flexbox/Grid).
- **Fuentes**: Google Fonts (Bebas Neue, DM Sans, DM Mono).
- **Hosting**: Cloudflare Pages.
- **Dominio**: Incluye 1 año de dominio .com o .cr.
- **Herramientas**: JavaScript (Vanilla) para animaciones y formulario.

##  🧩 Secciones del Sitio
- Hero: Presentación principal con estadísticas clave (10 días, $200, 100% diseño único).
- Servicios: Landing Pages, Presskits y Sitios de Negocio.
- Planes: Tres niveles de inversión con características detalladas.
- Portafolio: Proyectos reales con imágenes de muestra.
- Proceso: Flujo de trabajo en 4 pasos (Consulta → Propuesta → Desarrollo → Entrega).
- Contacto: Formulario integrado con WhatsApp.
